# ts-move-to-new-file

这是一个 TypeScript 和 React 项目。

## 安装

1. 克隆仓库：`git clone <repository-url>`
2. 安装依赖：`npm install`
3. 运行项目：`npm start`

## 使用

这是一个简单的示例，展示如何使用这个项目：

```tsx
import { main } from './components/main';

main();
```
